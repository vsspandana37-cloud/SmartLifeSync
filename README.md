# 🚀 SmartLife Sync Ultimate

SmartLife Sync Ultimate is a modern productivity and life management web application that helps users manage tasks, track habits, improve focus, and stay productive.

## 🌟 Features
- User Authentication
- Smart Task Management
- Habit Tracker
- Productivity Dashboard
- Pomodoro Focus Timer
- Water & Study Reminders
- Real-Time Weather Updates
- Motivational Quotes
- Real-Time Notifications

## 🛠 Technologies Used
- HTML, CSS, JavaScript
- Bootstrap
- Node.js & Express.js
- MongoDB Atlas
- JWT Authentication
- Socket.IO
- OpenWeather API
